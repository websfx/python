* 列表生成式
* 生成器都是迭代器 有两种方式
```buildoutcfg
1. (x * 2 for x in range(10))
2. def f():
    yield
    f()---->这个是一个生成器
    有两个方法 iter和next 
```
* 生成器在创建时就已经决定了能计算出值得个数
* 遍历时for循环做了三件事：1 调用iter方法 2 不断调用其next方法 3 处理stopiter异常
* log有5个级别 依次为 critical>error>warning>info>debug>notset
* 正则：匹配字符串 引入的目的是模糊匹配
* . 任意字符(除了换行符) 只能代指一个 通配符
* ^ 以什么开始  $ 以什么结尾
* * 重复匹配 .* 任意字符0次或多次[0,+oo]
* + 一次以上 即[1,+oo]
* ? 0次或1次 
* {n,m} 匹配多少次 即n到m次 
* 贪婪匹配
* *等于{0，+oo} +等于{1,+oo} ?等于{0,1}
* [] 或的关系，只会选择一个 取消特殊字符 但有例外 - ^这俩例外
* [^t] 取反 不匹配t [^45] 不匹配4和5 中间不用加逗号
* 反斜杠后边跟元字符去除特殊功能 所谓转义
* \d 数字 \D [^0-9] \s 任何空白字符 \S  任何非空白字符
* \w 任意字母数字字符 \W 任何非字母数字字符 \b 一个特殊边界 特殊字符的一个边界
* () 分组  | 或
* 正则表达式: findall() 返回一个列表 search() 返回一个对象 对象可以调用group方法
* match() 只在字符串开始匹配 split()分割  sub() 替换 compile() finditer() 返回一个可迭代对象
